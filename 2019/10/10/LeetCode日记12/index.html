18.四数相加

发表于 2019-10-10 阅读次数：

题目要求：

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

注意：

答案中不可以包含重复的四元组。

示例：

给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

满足要求的四元组集合为：
[
[-1, 0, 0, 1],
[-2, -1, 1, 2],
[-2, 0, 0, 2]
]

进行两次for循环进行头两个数字的确认，然后再从后面的数中选两个进行相加看是否等于target

这里有个有趣的点，因为最终答案是一个大列表里有不重复的多个列表，得用set，而set不能添加列表所以把一组结果换成（）元祖就能添加了。然后最后将set元素加到列表里
